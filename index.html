EV Score Components (0–100)
ComponentMax ptsWhat drives it
True probability30No-vig win prob from blended book lines. 50% = 0pts, 70% = 30pts
Edge vs consensus25Best available odds vs true prob. ≥8% edge = full 25pts
Book line spread20Disagreement across books. ≥15pt spread = full 20pts (signals sharp action)
Odds band15−100 to −150 = full 15pts. Heavy favs (< −200) and big underdogs penalized
Book count10≥4 books = 10pts, 3 books = 7pts, <3 = 3pts. More books = more reliable line
7-Tier Staking Rules
ScoreTierKellyStake rule
70–100Elite¼ KellyKelly stake. Floor = tier % if Kelly = $0
60–69Strong¼ KellyKelly stake. Floor = tier % if Kelly = $0
55–59Value⅛ KellyKelly stake. Floor = tier % if Kelly = $0
48–54Lean1/16 KellyKelly stake. Floor = tier % if Kelly = $0
40–47Marginal1/32 KellyKelly only, no floor. $0 = pass
35–39Borderline—Pass unless promo/token
0–34Avoid—Hard pass
Key Rules
· Odds sweet spot: −100 to −150 scores maximum 15/15 on odds band. Below −200 scores ≤8pts.
· Kelly floor: Lean tier and above always show a minimum stake even if Kelly math returns $0. The floor is the flat tier % stake labeled "tier".
· Kelly = 0 means: edge exists per the score, but the price is too short relative to the edge for Kelly to recommend a stake. Tier floor keeps it actionable.
· Method tags: K = Kelly stake used · C = Custom amount · T = Tier flat % used
